#### рҹ’Ў What It Does

ExplainReport is a **full-stack AI web application** that transforms the experience of reading medical lab reports. Users simply upload a PDF of their medical report, and the system does everything else вҖ” extracting, analyzing, and explaining every value in plain language.

#### рҹ”¬ Core Feature Breakdown

| Feature | Description |
|---|---|
| рҹ“„ **PDF Upload & Parsing** | Uses **Apache PDFBox** to extract raw text from any medical PDF вҖ” blood tests, metabolic panels, CBC reports, etc. |
| рҹ”Қ **Parameter Detection** | Automatically identifies medical parameters: Hemoglobin, WBC, RBC, Platelets, Glucose, Creatinine, and more |
| рҹ“Ҡ **Range Comparison Engine** | Compares every extracted value against established normal medical ranges and classifies as **Normal / Low / High** |
| рҹҡЁ **Risk Flag Detection** | Highlights critical deviations that may indicate health risks and flags them prominently for user attention |
| рҹӨ– **AI-Generated Summary** | Sends structured data to the **Groq API (LLM)** and gets back a clean, jargon-free paragraph explaining the report |
| вқ“ **Doctor Question Generator** | AI produces a list of intelligent, personalized questions to ask your physician at your next visit |
| рҹ—әпёҸ **Next Step Recommendations** | Suggests whether to monitor, retest, consult a specialist, or take immediate action |
| рҹ’¬ **Interactive AI Chat** | Users can ask follow-up questions about their specific report вҖ” "What does a low WBC mean for me?" |
| рҹ“қ **Export Results** | Download the full AI analysis as a formatted **PDF or TXT file** for sharing with a doctor |
| рҹ”’ **Privacy-First Design** | Zero report storage вҖ” all data is processed in-memory and discarded after the session |
