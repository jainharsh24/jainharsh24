### 🧠 &nbsp; AGENTIC EXPLAINREPORT

<img src="https://img.shields.io/badge/STATUS-ACTIVE-success?style=for-the-badge"/> &nbsp;
<img src="https://img.shields.io/badge/LANGUAGE-Java-ED8B00?style=for-the-badge&logo=openjdk&logoColor=white"/> &nbsp;
<img src="https://img.shields.io/badge/AI-Groq_LLM-FF4B00?style=for-the-badge"/> &nbsp;
<img src="https://img.shields.io/badge/TYPE-Agentic_AI_App-blueviolet?style=for-the-badge"/>

**[🔗 View Repository →](https://github.com/jainharsh24/AI-ExplainReport)**

An AI-powered web app that reads your medical lab report (PDF) and explains it in plain language — no medical degree required.

**✦ What it solves**
Most patients receive lab reports and can't interpret them. This bridges the gap between clinical data and patient understanding — making healthcare information accessible to everyone.

**✦ Features**
- 📄 Upload any PDF medical report and auto-extract all lab values
- 🔍 Detects parameters like Hemoglobin, WBC, Glucose, Creatinine, and more
- 🚦 Classifies every value as Normal / Low / High against medical ranges
- 🚨 Flags risk indicators that need immediate attention
- 🤖 Generates a plain-language AI summary via Groq LLM
- ❓ Suggests smart questions to ask your doctor at your next visit
- 🗺️ Recommends next steps — monitor, retest, or consult a specialist
- 💬 Interactive AI chat to ask anything about your specific report
- 📝 Export the full analysis as PDF or TXT
- 🔒 Privacy-first — zero data stored, all in-memory processing

**✦ Impact**
Turns a confusing stack of numbers into a clear, actionable health snapshot. Empowers patients to walk into doctor visits informed, not anxious.

`Java` · `Spring Boot` · `Apache PDFBox` · `Groq API` · `HTML/CSS`

### ✅ &nbsp; FLASK TASK MANAGER

<img src="https://img.shields.io/badge/STATUS-DEPLOYED-blue?style=for-the-badge"/> &nbsp;
<img src="https://img.shields.io/badge/LANGUAGE-Python-3776AB?style=for-the-badge&logo=python&logoColor=white"/> &nbsp;
<img src="https://img.shields.io/badge/FRAMEWORK-Flask-000000?style=for-the-badge&logo=flask&logoColor=white"/> &nbsp;
<img src="https://img.shields.io/badge/DB-SQLite-07405E?style=for-the-badge&logo=sqlite&logoColor=white"/>

**[🔗 View Repository →](https://github.com/jainharsh24/flask-task-manager)**

A clean, multi-user task management web app — built for people who just want to get things done without the bloat.

**✦ What it solves**
Most productivity tools are over-engineered. This is a focused, no-nonsense app where you log in, add tasks, update their status, and track your day — nothing more.

**✦ Features**
- 🔐 User registration and login with session-based authentication
- 📋 Personal dashboard per user — fully isolated task lists
- ➕ Add tasks instantly with zero friction
- 🔄 3-stage task lifecycle: `Pending → Working → Done`
- 🗑️ Bulk clear all tasks with one click
- 👤 Profile menu in header with username display and logout
- 🧱 Persistent SQLite storage via Flask-SQLAlchemy ORM
- ☁️ Heroku-ready deployment (Procfile included)

**✦ Impact**
A practical, deployed tool that demonstrates full-stack Python development — from auth to database to deployment-ready packaging.

`Python` · `Flask` · `Flask-SQLAlchemy` · `SQLite` · `HTML/CSS`
